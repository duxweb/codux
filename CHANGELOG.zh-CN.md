## [0.1.11] - 2026-04-17

### Changed

- 将 Git 面板自动刷新职责完全下沉到 Git store，应用层现在只负责面板生命周期，而仓库监听与刷新合并都保留在 Git 驱动内部。
- 调整 Git 面板视图，直接监听 Git store，让文件状态自动刷新与远端同步状态变化保持在同一条渲染链路中。

### Fixed

- 修复 Git 文件列表自动刷新问题，本地文件新建、删除以及 AI 生成的改动在面板打开时都会立即反映出来，不再需要手动刷新。
- 修复 Git 面板刷新时丢失选中文件和可见 diff 状态的问题，自动刷新后不再回退到过期或空白详情。
- 修复切换项目、窗口重新激活以及取消最小化后终端焦点恢复问题，Shell 不再需要额外点击一次才能继续输入。
- 修复 Git 文件行尾随操作按钮导致的行高抖动问题，右侧状态与操作区域现在保持布局稳定。
